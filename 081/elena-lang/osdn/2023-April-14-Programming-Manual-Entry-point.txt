=== エントリーポイント (プログラムの開始地点)

プログラムエントリは、プログラムの起動時(いくつかの準備の後)に呼び出されるメインプログラムコードです。最後のステートメントが実行されると、プログラムは終了します。メインコード (またはネストされたコード) で発生した例外は、システムハンドラによってキャッチされるため適切な (安全に) プログラムの終了が可能になります。

エントリポイントを作成するには、プロジェクトのルート名前空間に引数を持たないパブリック関数を '''program''' という名前で宣言する必要があります (最も単純な場合はプロジェクトのソースファイルに記述します)。関数の記述は、C 言語のようなプログラミング言語に精通しているかたであれば非常に簡単なことです。

{{{ code ruby

public program()

{

}

}}}

コードはカーリーブラケットの内側に記述され、ステートメントで構成されています。ステートメントはセミコロンで終了する必要があります (最後のステートメントを除き、セミコロンはオプションです)。一般的なステートメントは、メソッドの呼び出し (メッセージの送信)、各種処理または制御フローステートメントです。この用例では、 '''console''' シンボルでシステムコンソールにテキストを出力します。メッセージの引数は文字列リテラルです (ダブルクォートで囲みます)。

{{{ code ruby

public program()

{

   console.writeLine("Hello World!");

}

}}}

実行形式は、

{{{ code text

Hello World!

}}}
