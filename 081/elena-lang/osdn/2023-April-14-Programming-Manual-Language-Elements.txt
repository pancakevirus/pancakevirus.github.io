== 言語の構成要素

ELENA の学習は、基本的なプログラムの要素 (プログラム、変数、シンボル、クラス、制御フロー) に慣れることから始まります。

ELENA プログラムは、ソースファイル (拡張子 '''*.l''') とメインプロジェクトファイル (拡張子 '''*.prj''') で構成されています。プロジェクトファイルには、コンパイルが行われるソースファイルの関連情報とコンパイルの処理結果 (ライブラリまたは実行可能ファイル) に関する主要情報が含まれています。最も単純なプロジェクトタイプはコンソールあり、これはスタンドアロンのコンソールプログラムを生成します。

新しいプロジェクトを作成する最も簡単な方法は、統合開発環境 (IDE) の起動後に '''File->New->Project''' を選択します。

[[Embed(65635302-5c8e2a80-dfe0-11e9-9059-376a9ec779fb.png, float=checked)]]

Project の Type (プロジェクトの種別) には console を選択、ルートの Namespace (名前空間) には example1 を記入、そして Target file name (ターゲットの実行可能ファイル名) には example1 を指定します。

プロジェクトには少なくともソースファイルが一本含まれている必要があります ('''File->New->Source File''' から作成)。プロジェクトを保存すると、最初のプログラムを作成できるようになります。
