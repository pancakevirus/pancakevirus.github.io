### 数値リテラル

ELENA は以下の数値リテラルをサポートしています。

 ```ruby
   int  n1  := 2;             // 32bit 整数リテラル
   int  n2 := -23; 
   int  x1 := 0Fh;            // 符号あり 32bit 十六進数整数   
   uint x2 := 0FH;            // 符号なし 32bit 十六進数整数        
   long l1 := 1234567890123l; // 64bit 整数リテラル
   long l2 := -23354436l345l;
   real r1 := 23.2;          // 浮動小数点リテラル
   real r2 := 1.2e+11;
```
