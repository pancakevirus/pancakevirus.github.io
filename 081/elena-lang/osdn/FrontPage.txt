== ELENA とは

=== 関数型言語

ELENA は第一級関数とファーストクラスメッセージが使えます。

=== オブジェクト指向言語

ELENA はオブジェクト指向の豊富な機能として抽象クラス、インターフェース、シングルトン、クラス定数、名前付きコンストラクタ、可変長引数メソッドとコンストラクタ、およびクラス拡張が使えます。また、メソッドの再送信 とディスパッチなどの高度な機能も使えます。

=== 動的言語

動的言語の機能も使えます: 多重コンストラクタ / 多重メソッド、および ジェネリックメソッド。 ミックスイン、動的レコードなどを実装するために カスタムメッセージディスパッチャーが用意されています。

=== インタプリタ

実行時のコンパイル処理なしで 実行中にコードをビルドできます。

=== カスタマイズ性

自作のスクリプト文法を定義してランタイムで実行できます。

=== フリー

ELENA の使用許諾条件は MIT ライセンスです。どなたさまでも自由に使えます。ソースコードは GitHub で一般公開しています。

== 概要

ELENA は遅延バインディングに対応した汎用プログラミング言語です。関数型プログラミングとオブジェクト指向プログラミングの機能を兼ね備えたマルチパラダイム言語です。強い型付けと弱い型付け、ランタイム変換、プリミティブ型のボクシングとアンボクシング、外部ライブラリ関数のインタフェース (FFI) ができます。メッセージディスパッチを扱うために豊富なツールセットが用意されています : 多重メソッド、メッセージ資格付与、ジェネリックメッセージハンドラ。多重継承はミックスインと型のインターフェースでシミュレートします。組み込みスクリプトエンジンによりにカスタム定義のスクリプトをアプリケーションに組み込めます。 単体アプリケーションと仮想マシンのクライアントで使えます。

=== 主な機能

 * 自由なオープンソース (MIT ライセンス)

 * すべて揃ったソースコード

 * Unicode 対応 (UTF-8, UTF-16, UTF-32)

 * GUI による統合開発環境とデバッガ (ELENA IDE)

 * オプション型 (型推論)

 * 多重ディスパッチ / 多重メソッド

 * 複数値による返値

 * 静的メソッドの継承

 * 可変長引数メソッド

 * 生成可能メソッド

 * クロージャー (閉包関数)

 * ミックスイン

 * 型のインターフェースと変換

 * クラスとコードのテンプレート

 * スクリプトエンジン

== インストール方法

インストールしたいフォルダに zip ファイルの内容をすべて展開します。

必要に応じて、 bin フォルダのパスをシステムの環境変数(例えば <アプリケーションのルートフォルダ>\bin)へ追加、あるいはセットアップの実行可能ファイルを使います。

ELENA を使うには Microsoft Visual C++ 2017 再頒布可能パッケージをインストールしてください。

ファイルを開いて、プログラムやライブラリのコンパイル、またはデバッグをするには ELENA GUI IDE (<app root>\bin\elide.exe) か ELENA コマンドラインコンパイラ (\bin\elc.exe) を使います。　

ELENA IDE では File → Open → Open Project オプションを選択後に プロジェクトファイル (*.prj) を開きます。つぎに、 Project →Compile オプションでプロジェクトのコンパイルと Project-Debug でデバッグを行います。
