common

指定された名前でグローバルな配列または変数を作成します。
文法:
common variable
common array[]
common name[ maxValue ] [[...]] [, ... ]
common name[ minValue To maxValue ][[...]] [, ...]
common name = expression { , ... }

これらはグローバルスコープでアクセス可能であるため、 Sub または Function で使用時は Shared として宣言する必要はありません。global ステートメントでも同じ結果となります。Common が使えるのは Sub や Function の外側に限ります。Sub または Function ではなく、 Dim をお使いください。詳しい情報は Dim を参照してください。
用例:
' 配列の作成
common myGlobalArray[3 To 10, 5]

' グローバル変数の作成と代入を行います
common myGlobalVar = 100

shared コマンドを参照してください。
