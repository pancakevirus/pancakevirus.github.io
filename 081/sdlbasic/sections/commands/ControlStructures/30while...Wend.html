while ... wend

式が true であり続ける限りコードのブロックを実行します。

文法:
while
        commands
        {exit while}
wend

while ステートメントはループを行うコマンドのブロックに関する始点を定義します。
wend ステートメントはブロックの終点を定義します。あるいは、代わりに end while ステートメントも使えます。

用例:
setautoback(50)
i=0
while inkey <> k_esc ------------------------ ESC が押されるまでループします
    i=i+1
    prints(i)
wend ----------------------------------------------- wend または end while のどちらも使えます

exit while コマンドを参照してください。
