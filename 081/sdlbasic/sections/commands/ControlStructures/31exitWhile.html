exit while

Exit while ... wend ループにおいて、 wend | end while ステートメントに後続するコマンドからプログラムの実行を継続します。したがって、 exit while と wend | end while ステートメントの間にあるコマンドは無視します。
文法:
exit while

用例:
i=0
while i<100
    i+=1
    if i>50 then : exit while : end if  ' -------->     ループ終端がある下方向へ直行..
    prints(i)
end while
waitKey        '   <----------------------------------- ...こちらがループ終端

詳細情報は while ... wend コマンドを参照してください。
