for ... next

指定された回数によりコマンドのブロックを繰り返します。繰り返し処理は専用の変数値により制御されます。

文法:
for variable = start to finish {step increment }
        commands
        {exit for}
        {continue}
next

start から finish まで変数を increment でインクリメントしながら繰り返し処理を行います。step 値の省略時、繰り返しごとに変数は 1 インクリメントされます。このコマンドは for ステートメントと next ステートメントにあるループ区間の内容を繰り返しを実行します。オプション機能として next ステートメントの代わりに end for ステートメントを使えます。

用例:

' 1 ～ 10 までの数値を表示します
for i = 1 to 10
    print i 
next

exit for, continue および step コマンドを参照してください。
