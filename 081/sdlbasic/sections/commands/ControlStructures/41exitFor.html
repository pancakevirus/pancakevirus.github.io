exit for

Exit for ... next ループにおいて、 next | end for ステートメントに後続するコマンドからプログラムの実行を継続します。したがって、 exit for と next | end for ステートメントの間にあるコマンドは無視します。
文法:
exit for

sdlBasic は next ステートメントに後続する次行からプログラムの実行を継続します。
用例:
' 1 ～ 10 までの数値を表示します
for i = 1 to 20
    print i
    if i = 10 then : exit for : end if
next
print "完了 " & i & " 回繰り返し"

for ... next コマンドを参照してください。
