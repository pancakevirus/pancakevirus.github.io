continue

ループの先頭へ直行します。これは for...next と while...wend ループ専用です。
文法:
Continue

以下の continue コマンドに関する用例はループの先頭へ直行します。そのため、 ' print i' ステートメントは変数 'i' が 5 以上の場合に実行されません。
用例:
' 5 以上の数値ならば処理はしません。
For i = 1 to 10 ' <----
    If i > 5 Then '       |
                Continue ' -----     これはループの先頭へ直行します 
            End If
    Print i
End For

for...next および while...wend コマンドを参照してください。
