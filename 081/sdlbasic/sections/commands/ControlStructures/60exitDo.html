exit do

Exit do ... loop サイクルにおいて、 loop ステートメントに後続するコマンドからプログラムの実行を継続します。したがって、 exit do と loop ステートメントの間にあるコマンドは無視します。
 
 文法:
exit do

sdlBasic は loop ステートメントに後続する次行からプログラムの実行を継続します。


 用例:
n=0
do
    n=n+1
    printS( n)
    for m = 1 to 10
    printS("    "&m)
    next
    wait(100)
    if n>10 then
            exit do    ' -----------------> ループ終端がある下方向へ直行...
    end if
loop 
waitKey    '    <------------------------ ...こちらがループ終端

 do ... loop コマンドを参照してください。
