read()

インラインデータ構造から指定されたポインタの位置に従い Data 値を読み取ります。
文法:
read( {pointer} )

ポインタを渡さない場合、 read コマンドは現在の Data 値を読み取り後に次の Data 値へポインタを移動します。
用例:
data(1,2,3,4,5,6,7,8)
for i = 0 to 7
    fprints( read()& " - " ) 
next
prints(chr(13))
for i =0 to 20
    n=rnd(7)
    prints( read( n ) )
next
waitKey

data() コマンドを参照してください。
