open ... for ... as

ファイルストリームを開きます。

 文法:
open filename for mode as stream

filename には開きたいファイルを指定します。
 mode はファイルの処理方法をコンピュータに指定します。
stream は開きたいファイルの番号を指定します。この番号は他のコマンドでファイルを参照するときに渡します。
以下のモードのうちいずれかを mode へ指定することでファイルを開きます: input (入力), output (出力), append (追記)
用例:
' テキストファイルを開く
open  "myfile.txt" for  input as  1
