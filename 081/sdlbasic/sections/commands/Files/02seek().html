seek()

ファイル位置の探査: 指定されたファイルの位置へ読み取り・書き込み指示位置を移動します。

 文法:
seek( stream , position )

stream は以前に open コマンドで開かれたファイルです。
position は読み取り・書き込み指示位置の移動先です。この位置はファイル先頭からのバイト数で指定します。

用例:
open "myfile.txt" for input as 1
seek( 1,2 )
while eof(1)=0
fprints("loc:"+str$(loc(1))+" - ")
file input #1,a$
fprints(a$)
end while
close 1
waitKey

loc() コマンドを参照してください。
