writeByte()

ファイルに単バイトを書き込みます。

 文法:
writeByte( stream , position )

stream は以前に open コマンドで開かれたファイルです。
position は書き込み先の位置をバイト単位で指定します。この位置はファイル先頭からのバイト数で指定します。

用例:
open "myfile.txt" for output as 1
writebyte( 1,asc("s") )
writebyte( 1,asc("d") )
writebyte( 1,asc("l") )
writebyte( 1,asc("B") )
writebyte( 1,asc("a") )
writebyte( 1,asc("s") )
writebyte( 1,asc("i") )
writebyte( 1,asc("c") )
writebyte( 1,13 )
close 1

readByte()コマンドを参照してください。
