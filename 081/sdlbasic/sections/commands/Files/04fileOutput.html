file output #

ファイルストリームへ一行書き込みます。
文法:
file output # stream, expression {,...}

stream は以前に open コマンドで開かれたファイルです。
従来の print # コマンド構文も引き続き使用可能です。 

 用例:
' テキストファイルを開く
open "myfile.txt" for output as 1

' テキスト書き込みます。
file output  #1, "Line1" 
file output  #1, "Line", "2" 
file output  #1, "Line", (2+4)/2 

' ファイルを閉じます
close( 1 )

file input # コマンドを参照してください。
