mirrorImage()

mirrorImage は指定されたスロットにあるイメージを垂直・水平方向へ反転します。

 文法:
mirrorImage( slot , x , y )

slot はイメージをロードしたスロットの番号です。
x and y はイメージを水平・垂直方向へ反転するために true または false (0 あるいは 1) を指定します。

 用例:
loadimage("sprites.png",1)
pasteIcon( 10, 10, 1 ) 
mirrorImage( 1, 1, 0 ) 
pasteIcon( 100, 10, 1 ) 
mirrorImage( 1, 0, 1 ) 
pasteIcon( 10, 100, 1 ) 
mirrorImage( 1, 1, 1 ) 
pasteIcon( 100, 100, 1 )
