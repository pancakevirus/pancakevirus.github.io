mouseZone()

マウスポインタが指定された矩形領域内にあるか確認します。

 文法:
variable = mouseZone( x , y , width , hight )

マウスポインタが交差する領域にあれば True (1) を返します。

 用例:
bar(100,100,100+200,100+150)
while not key(k_esc)
    if mouseZone( 100, 100, 200, 150 ) then
        ink(rnd(rgb(255,255,255)))
        bar(100,100,100+200,100+150)
    end if
wend
