paper()

テキストの背景表示カラーを設定します。あるいは color の省略時はテキストの現在背景表示カラーを返します。

 文法:
paper( color )
variable = paper()

color は Uint32, 十六進数形式、または rgb() コマンドで変換された RGB 値で表されます。Uint32 形式で表記したカラーが返されます。

 用例:
paper( 16711680 )
printS("これは赤色の背景です")
printS( "背景表示カラー = Uint32 " & paper() )
paper( 0x0000ff ) 
printS("これは青色の背景です")
printS( "背景表示カラー = Uint32 " & paper() )
paper( rgb(0,255,0) )
printS("これは緑色の背景です")
printS( "背景表示カラー = Uint32 " & paper() )
