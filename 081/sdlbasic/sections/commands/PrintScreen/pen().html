pen()

テキストカラーを設定します。あるいは color の省略時は現在のテキストカラーを返します。

 文法:
pen( color )
variable = pen()

color は Uint32, 十六進数形式、または rgb() コマンドで変換された RGB 値で表されます。Uint32 形式で表記したカラーが返されます。

 用例:
pen( 16711680 )
printS("これは赤色のテキストです")
printS( "テキストカラー = Uint32 " & pen() )
pen( 0x0000ff ) 
printS("これは青色のテキストです")
printS( "テキストカラー = Uint32 " & pen() )
pen( rgb(0,255,0) ) 
printS("これは緑色のテキストです")
printS( "テキストカラー = Uint32 " & pen() )
