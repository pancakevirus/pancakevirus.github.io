command$()

コマンドラインから渡されたプログラムの引数を返します。パラメータの個数はプログラム側で変数として使われます。
文法:
numberOfArguments = command $ ()
argument = command $ ( n )

パラメータを指定せずに呼び出し、または引数の番号 n を返さなければコマンドラインから渡された引数の総数を返します。また、パラメータ n を指定して呼び出した場合、最初の引数は n = 1 から開始します。
ドル記号を省略して command() と書くことも可能です。

用例:
print( command())
for i =1 to command()
print( command( i ) )
