os

sdlBasic を実行しているオペレーティングシステム名の情報を文字列で返します。
文法:
variable = os

返すことのできる値は:
windows Window 95, Windows 98, Windows me/2000, または WindowsXP オペレーティングシステムで実行中の場合。
unix Linux オペレーティングシステムで実行中の場合。
osx Apple OSX マシンで実行中の場合。
fbsd FreeBSD オペレーティングシステムで実行中の場合。
netbsd NetBSD オペレーティングシステムで実行中の場合。
openbsd OpenBSD オペレーティングシステムで実行中の場合。
morphos MorphOS オペレーティングシステムで実行中の場合。
用例:
print( os )
