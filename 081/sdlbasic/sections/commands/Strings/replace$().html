replace$()

index を起点とした replace で source の文字列を置換後に、その結果を string として返します。あるいはドル記号を省略して replace() と書くことも可能です。
文法:
string = replace$( position , source , replace )
string = replace( position , source , replace )

用例:
a$="ciao ciao"
print("string: "+a$)
print( replace( a$,"bye bye" ) )
print( replace( 5, a$,"bye bye" ) )

replaceSubstr$() コマンドを参照してください。
