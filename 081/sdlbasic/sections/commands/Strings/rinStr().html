rinStr()

inStr() の逆方向です。文字列に存在する部分文字列の位置を返します。一致しなければ、 0 を返します。
逆方向 Instr 関数は、末尾から先頭方向へ検索します。

文法:
position = rinStr( string , substring )
 position = rinStr( start , string , substring )

start は文字列の検索開始位置を指定するためのオプションの開始値です。
用例:
a$ = "picture.png"
n = rinStr( a$, "." )
print mid( a$, 1, n-1)

inStr() コマンドを参照してください。
