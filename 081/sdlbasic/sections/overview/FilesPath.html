ファイルのパス

現在のフォルダがプログラムのパスになります。事実上、プログラムと同一ディレクトリにあるファイルをロードするには、ディレクトリを指定せずにファイル名だけを記述します。対照的に、ファイルがサブディレクトリにある場合 (例えば "media") は、ディレクトリ名の後にスラッシュ記号 ( / ) とファイル名 (media/myfile.txt) を記述します。Windows プラットフォームでも / 記号を使うことを認識して下さい。これは無修正で Windows から Linux へプログラム (さらにその逆も) を移植可能にするためです。Windows ではバックラッシュ ( \ ) 記号も使えますが、これは非推奨ですので使用は控えてください。
