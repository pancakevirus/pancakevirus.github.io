# beep

Ring the bell within your computer; a synonym for ```bell```.

## Synopsis

```basic
beep
```

## Description

The ```beep```-command rings the bell within your computer once. ```beep``` is a synonym for [```bell```](bell.html).

## Example

```basic
beep:print "This is a problem ..."
```
