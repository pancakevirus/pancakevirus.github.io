# cos()

Return the cosine of its single argument.

## Synopsis

```basic
x=cos(angle)
```

## Description

The ```cos```-function expects an angle (in radians) and returns its cosine.

## Example

```basic
print cos(pi)
```

This example will print ```-1```.
