# sqrt()

Compute the square root of its argument.

## Synopsis

to be written

## Description

The ```sqrt```-function computes the square root of its numerical argument.

## Example

```basic
for a=1 to 5
  print a,sqrt(a),a**(1/2)
next a
```

As you may see from the output, sqrt can be written as ```**(1/2)``` (or ```^(1/2)```) too.
